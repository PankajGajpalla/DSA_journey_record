# Complexity
- Time complexity:
O(n)

- Space complexity:
O(n)

# Code
```cpp []
class Solution {
public:
    vector<int> countBits(int n) {
        vector<int> ans(n+1, 0);

        for(int i=1; i<=n; i++){
            ans[i] = ans[i>>1] + (i&1); // remember this:) 
        }

        return ans;
    }
};
```
